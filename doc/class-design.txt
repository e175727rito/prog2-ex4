マップクラス
フィールド変数
    ・初期位置　string x_s, y_s;
    ・ゴール位置　string x_g, y_g;
    ・マップ情報を保存できる2次元配列 int[][] maps[x_g][y_g];
メソッド
    ・初期位置、ゴール位置を読み取るコンストラクタ Map(file_name)
        Map()コンストラクタではテキストファイルの1、2行目を読み込んで、x_s, y_s, x_g, y_g の値を更新する。
    ・マップの情報を読み取るメソッド set_maps()
        set_maxps()メソッドではテキストファイルの3行目以降を全て読み込んで、maps を更新する。

アクセサ
    ・初期位置やゴール位置を決めるsetter,それらを得るgetterメソッドが必要
